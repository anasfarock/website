# anasfarooq.com

Personal website. Plain HTML and CSS, no frameworks.

## Stack

- HTML + CSS
- Poppins (body), Italianno (signature) via Google Fonts

## Structure

```
index.html   # everything is here
```

## Running locally

Just open `index.html` in a browser. No build step, no dependencies.

## Deployment

Works with any static host — GitHub Pages, Vercel, Netlify, etc.
